1) What is React?
    React is a JavaScript library used for building user interfaces, especially interactive 
    and dynamic web applications. It was developed by Facebook (Meta).
    React is a open-source, component based frontend library which is responsible
    for view layer of the application.
    Main objective of react is to develop user interfaces that improves the speed
    of the apps.

1) A) Features of React:
        Virtual Dom
        JSX
        One Way Data Binding
        Component Based Architecture
        Resuable Components

2) What is JSX?
    JSX stands for JavaScript XML and it is an XML-like syntax extension to ECMAScript. 


3) What are Ref's in react?
    In React, refs (short for references) are a mechanism to access DOM nodes or React 
    elements directly and store mutable values that do not cause a component to re-render
    when updated.
